# PRD: AI-Native GHG Consulting Platform (AUI MVP)

## 1. 프로젝트 개요
- **프로젝트 명:** AI-Native GHG Consulting Platform
- **목표:** 기존의 정적인 온실가스 산정 툴을 넘어, 사용자와 상호작용하며 '컨설턴트' 역할을 수행하는 Agentic UI(AUI) 기반 플랫폼의 MVP 구현.
- **핵심 가치:** 
    - 전문 지식 없이도 산정 가능 (AI 전문 상담)
    - 데이터 입력 및 가공 자동화 (AI 데이터 에이전트)
    - 검증 대응 가능한 보고서 자동 생성 (AI 보고서 에이전트)

## 2. 주요 기능 (MVP 범위)

### A. AI 전문 상담 에이전트 (The Oracle)
- **기능:** GHG Protocol 및 관련 가이드라인 기반의 실시간 질의응답.
- **상세:**
    - `ghg-protocol-revised.pdf`, `CSES_Scope3_Guidebook.pdf` 등 사내 문서를 지식 베이스로 활용.
    - 사용자가 산정 방법론이나 조직 경계 설정에 대해 질문하면 근거 페이지를 제시하며 답변.
- **UX:** 대시보드 내 상시 노출되는 채팅 인터페이스.

### B. AI 데이터 입력 에이전트 (The Data Clerk)
- **기능:** 비정형 텍스트 및 파일을 분석하여 대시보드 데이터로 변환/입력.
- **상세:**
    - "본사 전기요금 50만원 나왔어"라고 입력하면 `Scope 2` 전력 항목으로 자동 분류 및 수치 입력.
    - 복잡한 엑셀/PDF 업로드 시 데이터 구조를 파악하여 `MainCalculator`에 필요한 값 매칭 제안.
- **UX:** 채팅창에 텍스트 입력 또는 파일 드래그 앤 드롭 -> AI의 제안 -> 사용자의 승인/수정.

### C. AI 보고서 에이전트 (The Reporter)
- **기능:** 현재 산정된 데이터를 바탕으로 분석 텍스트 및 보고서 초안 생성.
- **상세:**
    - "이번 달 배출량 분석해줘" 요청 시, 전월 대비 변동 요인 분석 및 텍스트 리포트 생성.
    - `ReportGenerator`와 연동하여 PDF 보고서 내의 '비고' 및 '분석 의견' 자동 작성.
- **UX:** 보고서 생성 모달 내 "AI 분석 포함" 옵션 제공.

## 3. 사용자 시나리오 (MVP)
1. **상담:** 사용자가 "합작 법인도 우리 배출량에 포함해야 하나요?"라고 질문 -> AI가 가이드라인 기반 답변 제공.
2. **입력:** "안산 공장 가솔린 사용량 500리터 추가해줘" -> AI가 배출원 분류 후 "Scope 1 - 이동연소 - 휘발유(L)"로 인식하여 입력을 대기시킴 -> 사용자 승인 시 DB 저장.
3. **분석:** "올해 배출량이 왜 작년보다 높아졌지?" -> AI가 데이터 분석 후 "생산 시설 확대로 인한 Scope 2 전력 사용량 15% 증가가 주된 원인입니다"라고 답변.

## 4. 기술 스택 및 요구사항
- **Frontend:** Next.js (현재 스택 유지), Agentic UI 컴포넌트 추가.
- **Backend:** Prisma, Supabase (현재 스택 유지).
- **AI/LLM:** OpenAI GPT-4o 또는 Claude 3.5 Sonnet (Function Calling 필수).
- **Knowledge Base:** RAG (Retrieval-Augmented Generation) 시스템 구축.

## 5. 성공 지표 (KPI)
- 사용자가 직접 데이터를 수동으로 입력하는 횟수가 50% 감소.
- 산정 과정 중 가이드북 파일을 직접 열어보는 횟수 감소.
- 보고서 생성에 소요되는 시간 단축.
